# minmax
> 对应 CSS 的 min-width/max-width/min-height/max-height 等最大，最小值相关的.

## usage

```tsx live
<View debug_ spy={20}>
  <View wp={5} auto minh={40} maxh={100} ovs>
    少年时，想碰到一个聂小倩，拼了性命爱一场，天亮前带着她的魂魄远走他乡。
  </View>
  <View wp={5} auto minh={40} maxh={100} ovs>
    少年时，想碰到一个聂小倩，拼了性命爱一场，天亮前带着她的魂魄远走他乡。
    青年时，想碰到一个白素贞，家大业大，要啥有啥，吃完软饭一抹嘴，还有人负责把她关进雷峰塔。
    中年时，想要一个田螺姑娘，温婉可人，红袖添香，半夜写累了，让她变回原形，加干辣椒、花椒、姜、蒜片爆炒，淋入香油，起锅装盘。
  </View>

  <View wp={5} auto minmaxh={[40,100]} ovs>

  </View>
</View>
```
