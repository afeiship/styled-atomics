# raw
> 对应 CSS 的 raw.

- color
- cursor
- background
- width
- height
- opacity
- size
- margin
- padding
- top
- right
- bottom
- left

## usage

```tsx live
<View debug debug_ wh={800} auto>
  <View padding="100px">PADDING</View>
  <View margin="100px">PADDING</View>
  <View margin="10px auto">10px auto</View>
  <View width="10%" auto>10px auto</View>
</View>
```
