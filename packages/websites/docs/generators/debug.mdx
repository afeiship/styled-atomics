# debug

> 调试的时候用到。

## usage

```tsx live
<View spy={20}>
    <View debug p={10}/>
    <View debug="red@1" p={10}/>
    <View debug="green@5" p={10}/>
    <View debug="@3" p={10}/>
    
    <View debug__ p={5} spy={10}>
        <View>Item1</View>
        <View>Item2</View>
        <View>Item3</View>
    </View>
</View>
```
