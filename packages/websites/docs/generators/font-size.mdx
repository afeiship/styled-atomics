# font-size
> 字体大小.

## usage

```tsx live
<View spy={20}>
    <View debug p={10} f={12}>
        道可道，非常道；名可名，非常名。 无名，天地之始，有名，万物之母
    </View>
    <View debug p={10} f={14}>
        道可道，非常道；名可名，非常名。 无名，天地之始，有名，万物之母
    </View>
    <View debug p={10} f={20}>
        道可道，非常道；名可名，非常名。 无名，天地之始，有名，万物之母
    </View>
</View>
```
