# position
> 对应 CSS 的 position.

## usage

```tsx live
<View debug debug_ relative abs_ wh={800} auto>
  <View t={0}>T0</View>
  <View r={0}>R0</View>
  <View b={0}>B0</View>
  <View l={100}>left: 100</View>
  <View trbl={200} bg={'#eee'}>trbl: 300</View>
</View>
```
