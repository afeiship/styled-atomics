# radius
> 容器圆角.

## usage
> 当前盒子/子级添加圆角

```tsx live
<View debug sp={30} y>
    <View radius={2} debug p={20} />
    <View radius={5} debug p={20} />
    <View radius={10} debug p={20} />
    <View radius={100} debug p={20} />
</View>
```
