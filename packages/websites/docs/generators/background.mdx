# background
> 对应 css 的 background 属性相关.

## usage
> 当前盒子/子级添加背景

```tsx live
<View>
    <View mb={10} bg="#007aff">Content.</View>

    <View bg_={'#eee'}>
        <div>item1</div>
        <div>item2</div>
        <div>item3</div>
    </View>
</View>
```
