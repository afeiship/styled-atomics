# status

> 对应 css 的 status 属性相关.

## usage

> disabled 等状态。

- disabled
- hidden

```tsx live
<View debug_ space={20} tc space={10} x strip>
  <View
    disabled
    wh={100}
    circle
    as="img"
    src="https://pic.rmb.bdstatic.com/7f7a8d7b247d3aa430010f10a5765239.jpeg"
  />
  <View
    hidden
    wh={100}
    circle
    as="img"
    src="https://pic.rmb.bdstatic.com/7f7a8d7b247d3aa430010f10a5765239.jpeg"
  />
</View>
```
