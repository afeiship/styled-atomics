# text

> 对应 css 的 text 属性相关.

## usage

> text 的相关样式: 下划线/删除线/加粗/普通/user-select: none;/white-space

- tdn
- tdu
- tdlh
- tl
- tc
- tr
- tj
- tlr\_
- ell
- fwb
- fsn
- usn
- wsn
- wsnw
