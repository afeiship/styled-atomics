# box-sizing
> 对应 css 的 box-sizing 属性相关.

## usage
> 当前盒子/子级添加背景

```tsx live
<View tc mb_={20} strip debug_>
    <View p={20} wh={150} bsc>Box with bsc</View>
    <View p={20} wh={150} bsb>Box with bsb</View>
</View>
```
