# background
> 对应 css 的 background 属性相关.

## usage
> 当前盒子/子级添加背景

```tsx live
<View tc mb_={20} strip bg_={'#ccc'}>
    <View bg="green">Green</View>
    <View>BOX</View>
    <View>BOX</View>
    <View debug bgn>BOX with: bgn</View>
</View>
```
