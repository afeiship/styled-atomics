# margin

> 对应 css 的 margin 属性相关.

## usage

> 当前盒子/子级添加背景

- auto
- maa
- m0a
- m0
- mt0
- mr0
- mb0
- ml0
- mt_
- mr_
- mb_
- ml_

```tsx live
<View debug_ spy={20}>
     <View w={200} auto>
        Width + margin-left/right: auto; 实现居中
     </View>
     <View w={200} m0a>
        Width + m0a; 实现居中, 与上面不同的是： margin: 0 auto;
     </View>
</View>
```
