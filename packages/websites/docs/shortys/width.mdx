# width

## usage

> 当前盒子/子级添加背景

- w100
- v100
- min100
- x
- max100

- f0

```tsx live
<View debug_ sp={20} strip x dib_ vam_ tc ovh>
     <View v100 x>
      100vw: v100 + x
     </View>

     <View wh={200}>
        <View w100 debug="@2">Box with w100</View>
     </View>
</View>
```
