# flex

> 对应 css 的 flex 属性相关.

## usage

> 当前盒子/子级添加背景

- wrap(true/false)
- row
- column
- align
- justify


## attr mapping

```js
// align/justify
{
  c: 'center',
  s: 'flex-start',
  e: 'flex-end',
  sa: 'sp-around',
  sb: 'sp-between',

  start: 'flex-start',
  end: 'flex-end',
  around: 'sp-around',
  between: 'sp-between'
}
```

```tsx live
<View mr_={10} debug_ dib_ tc strip>
  <View dflex row align="c" justify="sb" h={200} debug_>
    <View className="is-left">left</View>
    <View className="is-right">right</View>
  </View>
  <View dflex row align="c" justify="sa" h={200} debug_>
    <View className="is-left">left</View>
    <View className="is-right">right</View>
  </View>
</View>
```
