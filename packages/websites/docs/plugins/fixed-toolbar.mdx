### 插件名

```config
fixed-toolbar
```

### 安装

```bash npm2yarn
npm install @jswork/styled-plugin-fixed-toolbar
```

### 参数

| 名称  | 类型   | description                              |
| ----- | ------ | ---------------------------------------- |
| value | string | 方向(top/right/bottom/left; 默认: right) |

> 一种 fixed-toolbar 的列表布局。

### 原理


### 用法

```jsx
<View
  auto
  wp={8}
  plugin={[
    { name: 'fixed-toolbar', value: 100, gap: 20, role: 'header' },
    { name: 'fixed-toolbar', value: 50, gap: 10, role: 'footer' }
  ]}>
        <View className="is-header" bg="#007aff">
    Header
  </View>

  <View className="is-footer" bg="#4cd964">
    FOOTER
  </View>
</View>
```
