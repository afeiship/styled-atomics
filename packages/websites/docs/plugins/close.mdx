### 插件名

```config
close
```

### 参数

<!-- value: 'right', thick: 1, size: 3, color: 'gray' -->

| 名称  | 类型   | description              |
| ----- | ------ | ------------------------ |
| value | number | 占 size 大小的比例, 1-10 |
| thick | number | 默认 1，线条的厚度       |

> 绝对居中的一种实现方式。

### 原理

1. before/after 利用角度变化实现 close 的效果
2. em 实现大小的控制

### 用法

```jsx
<View plugin="close" c="darkgreen" />
<View plugin="close:6" circle bg="#eee" c="#000" />
```
