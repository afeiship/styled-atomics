### 插件名

```config
absolute-rect
```

### 安装

```bash npm2yarn
npm install @jswork/styled-plugin-absolute-rect
```

### 参数

| 名称  | 类型   | description                 |
| ----- | ------ | --------------------------- |
| value | string | 是否为水平/垂直居中(x/y/xy) |

> 绝对居中的一种实现方式。

### 原理
1. 父容器有高度，可以是100%实现的，可以是 flex:1 实现的；可以是height固定值实现的
2. 父容器 position:relative;
3. 当前插件: position: absolute; trbl 均会一个值，默认为0；并且设置 x:true/false

### 用法
```jsx
<View debug relative w={400} auto h={200} mb={20}>
  <View px={10} f={18} hlh={'50px'}>
    Y方向下的ScrollView
  </View>
  <View plugin="absolute-rect:0" t={50}>
    <View bg="#ddd" h={2000}>
      content
    </View>
  </View>
</View>
```
