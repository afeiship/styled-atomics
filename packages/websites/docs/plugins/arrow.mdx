### 插件名

```config
arrow
```

### 安装

```bash npm2yarn
npm install @jswork/styled-plugin-arrow
```

### 参数

<!-- value: 'right', thick: 1, size: 3, color: 'gray' -->

| 名称  | 类型   | description                              |
| ----- | ------ | ---------------------------------------- |
| value | string | 方向(top/right/bottom/left; 默认: right) |
| thick | number | 厚度                                     |
| size  | number | 箭头的大小                               |
| color | string | 颜色                                     |

> 绝对居中的一种实现方式。

### 原理

1. 设置一个上/右的边框
2. 旋转 45deg，以及倍数

### 用法

```jsx
<View dib>
  更多
  <View plugin="arrow:right" />
</View>
```

### 演示

```tsx live
<React.Fragment>
  <View db>
    更多
    <View plugin="arrow:right" />
  </View>

  <View db>
    <View dib mr={10}>
      展开查看更多
    </View>
    <View plugin="arrow:top" />
  </View>
</React.Fragment>
```
