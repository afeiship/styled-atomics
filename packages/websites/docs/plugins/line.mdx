### 插件名

```config
line
```

### 参数

| 名称  | 类型   | description |
| ----- | ------ | ----------- |
| value | number | 线条的宽度  |

> 绝对居中的一种实现方式。

### 原理

1. 利用 border-bottom 实现各种宽度的线条
2. todo: 后面会加上 hairline 的实现

### 用法

```jsx
<View plugin="line:1" my={10} />
```
