### 插件名

```config
radius-strip
```

### 安装

```bash npm2yarn
npm install @jswork/styled-plugin-radius-strip
```

### 参数

| 名称  | 类型   | description        |
| ----- | ------ | ------------------ |
| value | string | 在哪个位置去除圆角 |

## value

| 名称 | 类型   | description |
| ---- | ------ | ----------- |
| t0   | string | 上边        |
| r0   | string | 右边        |
| b0   | string | 下边        |
| l0   | string | 左边        |

### 原理

1. 利用 border-radius 的各种组合
2. 由于优先级的原因，结合了 important 实现的去除已有圆角的去除

### 用法

```jsx
<View sp={10} y>
  <View wh={100} bg="#333" radius="10" plugin="radius-strip:r0" />
  <View wh={100} bg="red" radius="10" plugin="radius-strip:r0b0" />
</View>
```
