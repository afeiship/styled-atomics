### 插件名

```config
radius-strip
```

### 参数

| 名称  | 类型   | description          |
| ----- | ------ | -------------------- |
| value | string | 在哪个位置去除圆角 |

## value

| 名称   | 类型   | description   |
| ------ | ------ | ------------- |
| center | string | 水平/垂直居中 |
| tr     | string | 右上角        |
| tl     | string | 左上角        |
| bl     | string | 左下角        |
| br     | string | 右下角        |
| top    | string | 上边框，居中  |
| right  | string | 右边框，居中  |
| bottom | string | 下边框，居中  |
| left   | string | 左边框，居中  |

### 原理

1. 利用 border-radius 的各种组合
2. 由于优先级的原因，结合了 important 实现的去除已有圆角的去除

### 用法

```jsx
<View space={10} y>
  <View wh={100} bg="#333" radius="10" plugin="radius-strip:r0" />
  <View wh={100} bg="red" radius="10" plugin="radius-strip:r0b0" />
</View>
```
