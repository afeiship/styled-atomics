### 插件名

```config
em-justify-list
```

### 参数

| 名称  | 类型   | description                              |
| ----- | ------ | ---------------------------------------- |
| value | string | 方向(top/right/bottom/left; 默认: right) |

> 一种 em-justify-list 的列表布局。

### 原理


### 用法

```jsx
<StyledBox dib>
  更多
  <StyledBox plugin="em-justify-list:right" />
</StyledBox>
```
