### 插件名

```config
responsive-list
```

### 安装

```bash npm2yarn
npm install @jswork/styled-plugin-responsive-list
```

### 参数

| 名称  | 类型   | description                 |
| ----- | ------ | --------------------------- |
| gap | numeric|string | 间隔 |
| value | numeric |每个item的最小宽度 |

> 绝对居中的一种实现方式。

### 原理
:::note
1. display: grid;
2. grid-template-columns: repeat(auto-fill, minmax(260px, 1fr));
3. 适用于兼容性要求不高的场景，如后台的list列表
:::

### 用法

> 仅 X 方向居中

```jsx
<View plugin="responsive-list:x">
  <div style={{width: 200, height: 400}}>我是要被居中的 element</div>
</View>
```
