### 插件名

```config
absolute-center
```

### 参数

| 名称  | 类型   | description                 |
| ----- | ------ | --------------------------- |
| value | string | 是否为水平/垂直居中(x/y/xy) |

> 绝对居中的一种实现方式。

### 原理

1. margin: auto
2. 上/下/左/右 均为 0

### 用法

> 仅 X 方向居中

```jsx
<StyledBox plugin="absolute-center:x">
  <div style={{width: 200, height: 400}}>我是要被居中的 element</div>
</StyledBox>
```

> 仅 Y 方向居中

```jsx
<StyledBox plugin="absolute-center:y">
  <div style={{width: 200, height: 400}}>我是要被居中的 element</div>
</StyledBox>
```

> XY 方向居中

```jsx
<StyledBox plugin="absolute-center:xy">
  <div style={{width: 200, height: 400}}>我是要被居中的 element</div>
</StyledBox>
```

## 代码
```jsx
import StyledBox from '@jswork/styled-box';
import styled from 'styled-components';
Object.assign(StyledBox.defaultProps, {
  styled,
  plugins: [require('@jswork/styled-plugin-absolute-center').default],
});

<StyledBox debug relative w={400} auto h={400}>
  <StyledBox
    plugin="absolute-center:xy"
    debug
    bg="#f60"
    style={{width: 200, height: 120}}
  />
</StyledBox>
```
